How to install haml

  • 因為haml是以Ruby的環境寫成
  • 所以我們得先下載Ruby
  • 選擇一個適合自己電腦的ruby檔案
  • 開啟cmd確認ruby是否有安裝成功
  • 再確認是否有gem

    • gem是ruby的第三方套件管理程式，類似於Centos環境中的yum、Ubuntu的apt-get、Python的pip

接下來找個自己喜歡的文字編輯器

就可以開始寫haml囉

  • 因為網頁都是以html呈現
  • 所以我們需要先把haml轉檔成html
  • 開啟cmd
  • 鍵入 haml index.haml index.html (index為檔案名稱)
  • 接下來在你的資料夾裡會有兩種檔案
    1. index.haml
    2. index.html
  • 雙擊index.html，就可以進入簡報囉>.0
